This directory contains the LaTeX2e support for the journal
"Decisions in Economics and Finance"

The following files are part of the macro package

  readme.txt    This file
  svjour.cls    The document class file for Springer journals
  usrguide.*    User's Guide (dvi, pdf)
  svdef.clo     The class option file for Decisions in Economics and Finance
  guidedef.*    User's Guide (tex, dvi, pdf) for Decisions in Economics and Finance
  template.tex  A fill-in-form for a standard article with usage comments

Run the *.tex files with your normal LaTeX command to get
the documentation for the package.
