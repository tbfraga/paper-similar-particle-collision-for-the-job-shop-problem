This directory contains the LaTeX2e support for the
journal "Inventiones mathematicae"


CONTENTS

  readme.txt    This file
  svjour.cls    Core document class file for Springer journals
  usrguide.*    Main User's Guide (PDF and DVI)
  svinvmat.clo  Class option file for Inventiones mathematicae
  invmat.*      User's Guide for Inventiones mathematicae (PDF and DVI)
  template.tex  A fill-in-form for a standard article with usage comments


INSTALLATION

Copy svjour.cls and svinvmat.clo to a place where LaTeX can find them.


GETTING STARTED

You should first read the "Main User's Guide" to get an overview
and then the "User's Guide for Inventiones mathematicae" for special
instructions concerning this journal. Alternatively to reading the
entire documentation, it is also possible to proceed by merely filling
out the template and then looking up additional commands as needed.
